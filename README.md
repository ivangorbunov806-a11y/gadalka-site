# Мистическая Гадалка

Одностраничный мобильный русскоязычный сайт для быстрого теста идеи гадания.

## Что внутри
- Мистический тёмный UI, мягкие свечения, карточный премиальный стиль.
- Сценарий из экранов: старт → выбор темы → выбор одной из трёх карт → магический процесс → результат → финал.
- Встроенная клиентская аналитика через `trackEvent()` (без внешних API).
- Вся логика и данные захардкожены в `app.js`.
- Понятная заглушка для VK-ссылки: `VK_PLACEHOLDER_URL` в начале `app.js`.

## Быстрый запуск
Откройте `index.html` напрямую в браузере

или поднимите локальный сервер:

```bash
python3 -m http.server 4173
```

После запуска откройте:

`http://localhost:4173`
